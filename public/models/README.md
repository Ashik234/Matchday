# 3D Models

## wc-trophy.glb

FIFA World Cup trophy used by the bracket section.

**Source:** Place a CC0 / freely-licensed GLB at `wc-trophy.glb` in this
directory. Suggested sources:

- https://www.meshy.ai/tags/worldcup (CC0)
- https://3dmodels.org/3d-models/fifa-world-cup-trophy/
- https://sketchfab.com (filter to CC0 or CC BY)

If the file is missing or fails to load, the bracket falls back to the
inline SVG trophy automatically (no crash).

To verify after dropping the file, hard-reload the bracket section and
check the network panel for `200 /models/wc-trophy.glb`.
